# text-analyzer

text-analyzer é um aplicativo Java simples que permite buscar uma **frase** e suas **palavras individuais** em qualquer página web.  

Ele faz:

- Solicita um URL ao usuário  
- Solicita uma frase para análise
- Busca no conteúdo da página a quantidade de vezes que a frase aparece
- Conta a quantidade de vezes que cada palavra da frase aparece  

O projeto utiliza:

- Java 17  
- Apache HttpClient 5
- [Jsoup](https://jsoup.org/) para extração de texto de páginas HTML
